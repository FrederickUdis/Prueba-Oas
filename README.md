# Aplicacion Oas Futbol

es una aplicacion capaz de agregar equipos con los siguientes atributos:
-Nombre del equipo
-Color de ropa
-Numero del equipo
-Numero de goles
-Puntos ganados
-Partido Asignado

## Comenzando 🚀

Para obtener una copia del proyecto solo es necesario clonar al git

### Pre-requisitos 📋

Angular
FireBase
